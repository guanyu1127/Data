功能介紹：這是一個公車數據分析網頁，使用 Gradio 建立介面，允許用戶上傳 PDF與輸入分析指令，然後透過 Google Gemini API 進行數據分析。程式會產生建議，並將結果輸出為 PDF 報告。最終，使用者可下載包含原始數據與 AI 建議的報告，幫助優化公車運行與管理。
